06 — Usage Rules
Blue (#0B5DD0) — Use For
Primary CTAs & buttons
Section headings accent word
Active nav link
Stat bar background
Focus rings on inputs
Top border accent on featured cards
Divider lines (3px accent bars)
Red (#FF4026) — Use For
Eyebrow / label text above headings
Accent words in hero headings
Stat number accents ("+")
Icon highlights
Secondary CTA (alternate pages)
Hover state accents
Badge backgrounds (with tint)
Tag lines & decorative accents
Dark (#17171B) — Use For
Body copy only
Headings (paired with accent)
Footer text on white
Dark CTA button (sparingly)
Never as full-page backgrounds
White & Grays — Use For
#FFFFFF — Card & page base (primary)
#F5F6F8 — Section alternating fills
#E5E8EC — Card borders, dividers
#CCD2DB — Input default borders
#8E99A9 — Placeholder, captions
#4E5A6C — Secondary body text
